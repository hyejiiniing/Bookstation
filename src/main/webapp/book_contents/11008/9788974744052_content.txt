제1부 좋은 플롯이란 어떤 것인가

시작하는 말: 플롯은 유기적인 작업 과정
일상의 이야기가 플롯의 순수한 표본 / 플롯은 창작의 첫 단계

- 플롯에 대한 오해와 이해
플롯은 작품의 원자들을 엮는 힘 / 플롯의 역동성 / 플롯은 작품의 방향을 잡아주는 나침반 /
플롯의 창작에는 법칙이 있다 / 기발한 착상보다 패턴이 중요하다 / 플롯은 몇 가지나 될까 /
<숨이 막힌 도베르만>과 <고래와 어부>의 차이 / 플롯이 없으면 긴장도 없다 /
<두 영국 신사> 이야기 / 아리스토텔레스가 주장한 기본 원리

- 좋은 플롯의 여덟가지 원칙
전제, 법칙을 깨고 싶으면 법칙을 배워라 / 첫째, 긴장이 없으면 플롯은 없다 /
둘째, 대립하는 세력으로 긴장을 창조하라 / 셋째, 대립하는 세력을 키워 긴장을 고조시켜라 /
넷째, 등장인물의 성격은 변해야 한다 / 다섯째, 모든 사건은 중요한 사건이 되게 하라 /
여섯째, 결정적인 것을 사소하게 보이도록 하라 / 일곱째, 복권에 당첨될 기회는 남겨두라 /
여덟째, 클라이맥스에서는 주인공이 중심적 역할을 하게 하라

- 플롯의 강한 설득력과 추진력
모든 작가가 같은 방법을 사용하지는 않는다 / 포르자와 포르다 / 몸의 플롯 /
마음의 플롯 / 쓰리 스투지가 보여주는 인생의 의미 / 어떤 플롯을 선택할 것인가

- 심연 구조를 획득해야 한다
심연 구조의 핵심은 도덕성 / 이분법은 작품을 망친다 /
등장인물을 난처한 상황에 처하게 하는 법 / 옳은 일은 그르고 그른 일은 옳다

- 등장인물과 플롯의 관계
등장인물은 3명이 적당하다 / 등장인물 간의 관계가 제대로 펼쳐져야 한다 /
행동 없이 인물 없고, 행동 없이 플롯 없다




시작하는 말: 플롯은 공공 자원이다
오래된 플롯도 효과적일 수 있다 / 위대한 작가도 아이디어를 빌려온다

- 추구: 돈 키호테는 사랑을 얻을 것인가
찾고 떠나고 추적하고 / 추구를 위한 여행 / 추구 플롯의 구조, 결단에서 시작 /
변화와 결단 / 주인공의 친구들 / 장애물을 등장시켜 흥미를 높여라 /
주인공의 내면적 변화에 충실하라 / 주인공은 깨달음을 통해 성숙해진다 / 점검사항

- 모험: 초점을 여행에 맞추어라
관객도 주인공과 함께 모험을 즐길 수 있게 하라 / 모험 플롯의 전형은 동화 /
등장인물을 흥미로운 상황에 처하게 하라 / 여행의 시작 / 세부묘사가 설득력을 결정한다 /
생생하게 묘사하라 / 점검사항

- 추적: 도망자의 길은 좁을수록 좋다
가장 흥미로운 게임은 추적이다 / 추적 플롯을 활용한 작품들 / 긴장감을 유지하라 /
추적의 플롯을 추적하기 / 상투성 극복하기 / 제한된 공간을 활용하라 / 점검사항

- 구출: 흑백논리도 설득력이 있다
희생자를 둘러싼 프로타고니스트와 안타고니스트의 대결 / 좋은 사람 대 나쁜 사람 /
약자에서 강자로 성장하는 프로타고니스트 / 프로타고니스트의 영원한 파트너 안타고니스트 /
그림자 같은 존재, 희생자 / 이별 다음에 추적 / 대역전 / 점검사항

- 탈출: 두 번 실패한 다음 성공하게 하라
탈출 이외의 다른 길은 없게 하라 / 주인공이 잡혀가야 이야기가 시작된다 /
탈출을 시도하지만 번번이 실패한다 / 두 번 실패한 후 탈출에 성공한다 / 점거사항

- 복수: 범죄를 목격하게 만들면 효과가 커진다
범죄가 끔찍할수록 복수는 호응을 얻는다 / 눈에는 눈 이에는 이 /
범죄로부터 이야기가 시작된다 / 복수 계획과 추적이 두 번째 극적 단계 /
세 번째 극적 단계는 대결 / 지나친 복수 / 정의의 보안관, 전형적 인물 /
햄릿, 고뇌하고 복수하고 그리고 죽는다 / 현대의 복수, 정의감 충족 /
복수의 대가 / 점검사항

- 수수께끼: 가장 중요한 단서는 감추지 않는다
수수께끼는 독자에게 차원 높은 도전을 요구한다 / 수수께끼는 시험이다 /
수수께끼의 변형, 미스터리 / 독자에게 적절한 단서를 제공하라 /
결정적 단서는 평범하게 보여야 한다 / 어떻게 할 것인가 /
첫번째 극적 단계에서 미스터리가 제시된다 / 다이아몬드는 평범한 돌 밑에 숨겨라 /
새롭고 치밀한 방법으로 미스터리를 해결한다 / 독자들은 미스터리가 해결되기를 기대한다 /
점검사항

- 라이벌: 경쟁자는 상대방을 이용하다
사자와 사자보다 사자와 여우의 대결이 더 흥미롭다 / 문학 속의 라이벌 /
라이벌 플롯의 대표적 고전, 벤허 / 갈등의 극적 세 단계 /
서로의 힘이 대등할수록 경쟁은 치열해진다 / 점검사항

- 희생자: 주인공의 정서적 수준을 낮게 하라
관객보다 뛰어난 주인공은 곤란 / 관객은 동일시할 수 있는 대상을 원한다 /
희생자 플롯의 전형, 신데렐라 / 두 번째 극적 단계 / 마지막 극적 단계 /
관객은 착한 사람이 행복해지기를 바란다 / 점검사항

- 유혹: 복잡한 인물이 유혹에 빠진다
유혹에는 치명적인 대가가 따른다 / 주인공이 유혹에 지게 만든다 /
유혹에 빠진 영향은 지속된다 / 세 번째 극적 단계, 참회와 용서 /
유혹의 이야기, <파우스트> / 관객은 유혹에 빠지는 인물을 동정한다 /
점검사항

- 변신: 변하는 인물에는 미스터리가 있다
변신의 플롯은 변화의 플롯이다 / 사랑은 저주를 치료하는 특효약 / 저주의 양상은 변신 /
세 가지 극적 단계 / 저주에는 금기사항이 있다 / 두 사람의 관계는 저주 해방의 열쇠 /
점검사항

- 변모: 변화의 책임을 누가 질 것인가
변모의 플롯은 변화의 본질에 집중한다 / 인간의 가슴에는 자기도 알지 못하는 면이 있다 /
<피그말리온>에서의 변모 / 사소한 일이 인생을 송두리째 바꾸기도 한다 / 불을 지피는 사건 /
점검사항

- 성숙: 서리를 맞아야 맛이 깊어진다
봄은 가을을 기다린다 / 주인공을 시험에 들게 하라 / 세상을 대면하는 어려움과 충동 /
사건이 일어나기 전에 작품이 시작된다 / 먹구름이 몰려오듯 사건이 일어난다 /
주인공의 믿음체계는 무너진다 / 어린이의 인식 수준에 맞춰 작가의 유년 시적을 보여줘라 /
점검사항

- 사랑: 시련이 클수록 꽃은 화려하다
진정한 사랑은 시련 속에 꽃 핀다 / 오르페우스와 에우리디케의 얄궂은 운명 /
장애 해결을 위한 좌절, 삼세번의 원칙 / 불행한 결말의 사랑 이야기 /
관객은 해피엔딩을 바란다 / 독자들의 동화를 위해서는 전형성이 중요하다 /
대중의 욕방을 인지하라 / 내용보다 표현 방법이 중요하다 / 진정한 정서를 표현하기 /
사랑은 좋지만 괴로움을 싫다 / 점검사항

- 금지된 사랑: 빗나간 열정은 죽음으로 빚을 갚는다
세상에서 가장 강한 것은 사랑이다 / 고전 속의 금지된 사랑들 /
사랑은 진실해도 간통의 대가는 혹독하다 / 누구나 동정도 받을 수 없는 악의 꽃 근친상간 /
시대에 따라 사회의 태도가 달라지는 동성애 / 봄과 겨울의 로맨스 /
금지된 사랑의 세 가지 극적 단계 / 점검사항

- 희생: 운명의 열쇠가 도덕적 난관을 만든다
등장인물이 치르는 희생 / 감동적인 희생, 스탠리 크레이머의 <하이 눈> /
희생은 커다란 개인적 희생과 함께 찾아온다 / 희생의 플롯으로 빛을 본 <카사블랑카> /
장소의 제한을 통해 긴장을 강조 / 등장인물의 변화가 주는 재미 /
인물의 변화과정은 설득력이 있어야 한다 / 희생 이외의 다른 선택이 없는 상황으로 몰고 간다 /
희생은 자기 보전의 본능에 저항하는 일 / 희생의 대가와 영향을 고려해야 한다 / 점검사항

- 발견: 사소한 일에도 인생의 의미가 담겨 있다
시간은 흘러도 인간의 본질은 변하지 않는다 / 작은 발견이 인생을 바꾸기도 한다 /
가능한 한 이야기를 늦게 시작하라 / 등장인물의 싸움을 중요하게 부각시켜라 /
<여인의 초상>, 등장인물이 끌고 가는 발견의 플롯 / 점검사항

- 지독한 행위: 사소한 성격 결함이 몰락을 부른다
누구나 지독한 행위를 당할 수 있다 / 지독한 행위의 플롯은 등장인물을 극단까지 몰고 간다 /
가장 지독한 행위는 오셀로의 질투 / 이아고보다 나쁜 녀석은 없다 /
지독한 행위와 질투심이 빚은 비극 / 등장인물을 동정하게 만들어라 /
등장인물은 별난 상황에 처한 보통사람 / 지독한 행위의 결말, 파괴 혹은 재생 / 점검사항

- 상승과 몰락: 늦게 시작하고 일찍 끝을 맺는다
상승과 몰락의 주인공은 자석과 같다 / 도덕적 난관에 빠져든 등장인물 /
인간 심리 탐구와 세부묘사가 중요 / 상승의 플롯은 인간 성격에 대한 이해가 기반한다 /
악마가 마음속에 숨어 있는지 그 누가 알라 / 등장인물이 거쳐 가는 발전 단계 이해하기 /
점검사항

플롯은 끊임없이 변할 수 있는 공작용 점토 / 플롯은 작가의 아이디어가 채택하는 형식 /
