머리말 언밸런스로 삶의 도약을 꿈꿔라

1부 [자기 분석하기] 나를 먼저 알고 인생을 설계하라

1장 인생에도 기본기가 필요하다

1. 언밸런스의 파워를 깨우쳐라
언밸런스가 역량자를 만든다 / 기본은 주인의 삶을 선택하는 것이다 / 갇히지 않은 열린 마음을 가져라 / 아마추어와 프로의 차이를 깨달아라

2. 나만의 성공 기준을 마련하라
남의 기준이 내 기준이 돼서는 안 된다 / 기준이 명확한 성공 목표를 가져라 / 셀프 업그레이드로 몸값을 올리는 게 재테크다 / 자테크로 성장의 순간을 경험하라 / 내일 열매를 맺으려면 오늘 씨앗을 뿌려라

3. 대안 없는 문제 제기가 나를 망친다
삶을 더 나은 방향으로 설계하고 실천하라 / 세상을 어떻게 바라보는지가 중요하다 / 위기의식을 가진 긍정론자가 돼라 / 좋은 태도는 매일 연습해야 얻어진다

2장 인생의 로드맵을 짜라

1. 크레이지 아이디어를 가져라
나만의 성공 스토리를 만들 ‘미친 생각’을 하라 / ‘보이지 않는 것’을 보는 사람이 한계를 넘어선다 / 인생 경영에도 원대한 비전이 필요하다 / 현실은 어렵지만 미래는 밝다

2. 인생 로드맵으로 구체화하라
구체적인 계획이 실현 가능성을 높인다 / 앞으로 30년, 인생 로드맵으로 준비하라 / 현재를 깊게 멀리 보면 미래를 통찰할 수 있다 / 가슴이 웅장해지는 경험으로 그릇을 키워라

3. 실천에도 기술이 필요하다
지루함을 견디게 해줄 작은 성취를 반복하라 / 스마트하게 계획하고 실천하고 점검하라 / 오늘은 오늘의 과업에 집중하라

2부 [성공 선순환 구조 만들기] 나만의 성공 선순환 구조를 짜라

3장 작은 성공으로 성공하는 법을 배운다

1. 작은 성공이 나를 키운다
쉬운 것부터 공략해서 성공 사이클을 완성하라 / 작은 성공이 쌓여 큰 성공을 이룬다 / 타인의 실패와 나의 성공이 성장 자산이 된다

2. 각자에게는 각자의 마중물이 있다
리더의 통찰을 나의 것으로 업그레이드하라 / 성공의 원동력이 되는 마중물을 마련하라 / 절박함은 마중물을 만들 확률을 높인다

3. 도전을 성장의 변곡점으로 만들어라
기록의 힘은 세다 / 선택은 기억에서 시작된다 / 기록은 반복되는 실패를 예방한다

4장 큰 성공으로 도약하는 법을 찾는다

1. 스스로에게 기회를 주라
시작은 시작일 뿐이다 / 스스로에게 끊임없이 기회를 주라 / 도전 의식은 기회의 문을 여는 열쇠다

2. 도움닫기가 티핑 포인트를 만든다
기대보다 준비로 임계점과 변곡점을 맞이하라 / 도움닫기 후에 도약할 수 있다 / 가장 절망할 때 기회가 열린다

3. 힘들 때 가장 많이 성장한다
가장 힘든 시기에 가장 많이 성장한다 / 기회가 왔을 때는 바닥부터 싹 바꿔라 / 목표 수립-실행-반성이라는 루틴의 힘을 활용하라

3부 [내재 역량 키우기] 포기하지 않는 꾸준함이 곧 힘이다

5장 자신을 긍정하는 경험주의자가 돼라

1. 나를 믿고 포기하지 않으면 된다
믿음이 변화를 가져온다 / 긍정적 자세가 지혜를 얻게 한다 / 완벽함보다 꾸준함이 실력을 키운다

2. 마음 밭을 가꿔야 결실이 맺힌다
마음은 눈에 보인다 / 평소의 태도가 인생의 궤도를 결정한다 / 좋은 마음 밭을 가꾸는 게 우선이다

3. 자기 성장을 믿는 경험주의자가 돼라
꽃길보다 가시밭길에서 더 좋은 경험을 얻는다 / 노력했다면 성과가 따라올 시간을 주라 / 경험이 역량을 키우는 훈련이자 리더십 교육이다 / 역경의 총합이 역량이 된다

6장 불안은 일과 자신에 대한 성찰에 도움이 된다

1. 일을 해도 불안하다면 지금에 충실하라
취업은 또 다른 불안의 시작이다 / 큰 산에 오르는 게 더 의미 있는 일이다 / 불안할 때가 답을 찾을 절호의 기회다

2. 적성은 찾는 게 아니라 만드는 것이다
적성에 맞지 않아도 적응하면 된다 / 각각의 능력을 통섭적으로 확대해 가라 / 적성이 아닌 일에서 가능성을 확인하라

3. 파이어 말고 질적 균형에 집중하라
파이어 후에도 자아실현의 욕구는 존재한다 / 질적 균형에 집중하라 / 성장하지 않는 인간은 불행하다

7장 멈출 때와 가야 할 때를 가릴 줄 알아야 한다

1. 멈출 때를 아는 것도 능력이다
비용의 함정에 빠지지 마라 / 딜레마존에서 과감하게 움직여라 / 변화의 흐름을 살피고 유연한 관계를 형성하라

2. 위기의 순간에 멈추고 생각하라
인생은 길게 보고 문제는 깊게 생각하라 / 중요한 일에 집중할 에너지를 확보하라 / 지금이 에너지를 충전할 기회다

3. 고민은 하되 걱정은 말자
고민과 걱정을 구분하라 / 통제력을 갖고 있는 문제만이 고민의 대상이다 / 여유가 최고의 조언자다

4부 [업무 역량 키우기] 일잘러와 리더가 되기 위한 역량을 길러라

8장 셀프 업그레이드로 일잘러가 돼라

1. 상사의 의도를 먼저 파악하라
관찰과 경청은 시행착오를 줄여준다 / 스스로의 한계부터 인정하고 상사의 의도를 파악하라 / 수업과 학습으로 한계를 넘어서라 / 상사를 주도하면서 이끌 수 있어야 한다

2. 업무의 질을 어떻게 높일 것인가
틀에 갇히지 말고 업그레이드하라 / 능숙한 것에 익숙해지는 것을 경계하라 / 이해도를 높이려면 입체적으로 볼 줄 알아야 한다 / 감정노동에서 승점을 얻어라

3. 경쟁과 고객이 나를 업그레이드한다
글로벌 스탠더드에 맞춰라 / 일류 고객이 일류 직원을 만든다 / 자리가 바뀌면 역할도 바뀌어야 한다

9장 솔직하고 유연한 게 리더의 덕목이다

1. 보고 배울 수 있는 자리에 선다는 것
상사의 자격을 고민할 때 리더십을 배운다 / 리더는 휴지통을 설치하는 사람이다 / 닮고 싶은 리더의 리스트를 작성하라

2. 솔선수범과 기다림으로 리더십을 발휘하라
말의 힘을 느끼고 싶다면 반드시 행동하라 / 리더를 믿게 하는 것은 일관된 행동과 메시지다 / 독려하고 기다리는 게 리더십이다

3. 틀린 것을 지적할 때도 매너가 필요하다
거절하는 경험도 중요하다 / 공평이 아니라 공정이 나와 조직을 키운다
공감하고 인정하고 지혜를 나누라
솔직하되 과하지 않게 소통하라

5부 [휴먼 자본 축적하기] 성공의 관계망을 갖춰라

10장 관계 스트레스를 줄이는 최상의 솔루션을 찾아라

1. 사람 때문에 회사를 그만둘 수는 없다
관계 스트레스는 누구에게나 있는 상수常數다 / 자신만의 갈등 대처 지침을 만들라 / 갈등에 매몰되지 말고 자신의 길을 가라

2. 진심을 전하는 데도 스킬이 필요하다
역지사지로 바라볼 줄 알아야 한다 / 잘못을 지적할 때는 일과 사람을 분리하라 / 듣기 실력을 키워 마음을 얻어라

3. 인정, 칭찬, 솔직함으로 이끌라
올라갈수록 관계의 구심력이 중요하다 / 인정과 칭찬이 가장 좋은 선물이다 / 신뢰가 최상의 팀워크를 만든다

11장 아름답게 조직을 떠나라

1. 누구나 떠나야 할 순간이 있다
다시 비상할 날이 반드시 온다 / 현실주의자와 패배주의자를 구분하라 / 반성과 성찰을 위한 후회를 하라

2. 무례한 퇴사는 직무유기다
모든 경험을 자산으로 삼아라 / 무조건 책상부터 비우지 마라 / 순서를 지키고 예의를 갖추고 떠나라

3. 최소한 사람은 남기려 노력하라
다음 만남을 기약하며 먼저 호의를 베풀라 / 능력 있는 사람은 많지만 믿을 만한 사람은 드물다 / 좋은 이별로 사람을 남기는 노력을 하라

6부 [지속성장하기] 끝은 없다

12장 재도전의 기회도 준비된 자에게 온다

1. 심장이 두근거릴 때 다시 인생을 설계하라
두려움도 필요에 의해서 만들어진다 / 재도전일수록 바닥부터 시작한다고 마음먹어라 / 다시 길게 설계하라

2. 전 생애에 걸쳐 전문가로 성장하라
전문성을 확장하면 다방면에 경쟁력이 생긴다 / 비전이 늘 새롭게 만든다 / 끊임없이 업그레이드할 기회를 찾아라

3. 도전의 법칙은 여전히 유효하다
필요한 것은 결국 실력이다 / 잘할 수 있는 일과 하고 싶은 일의 교집합을 찾아라 / 1만 시간의 법칙을 거스를 축적의 힘을 키워라

나가며 배울 수 있는 시기에 제대로 배우는 것이 중요하다
