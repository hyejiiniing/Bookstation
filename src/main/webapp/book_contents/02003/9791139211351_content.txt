PROLOGUE

1장 책 읽기가 너무 힘들어요
저는 글이 잘 안 읽혀요 / 독서를 실천하지 못해 늘 후회만 해요 / 책을 잘 읽는 효과적인 방법이 있을까요? / 책만 잡으면 잠이 와요 / 저는 난독증이 있는지 책을 읽기가 어려워요 / 아이들에게 책 읽어 주는 게 잘 안되네요 / 더워서 책이 손에 잘 잡히질 않네요 / 책은 폈는데 진도가 안 나가요

2장 독서는 취미 활동이 아니에요
책을 많이 읽어도 포용력이 없는 사람은 왜 그런가요? / 모든 상황에서 이상하게 화를 참을 수가 없어요 / 매일 책 읽으라는 소리만 하니 답답해요 / SNS로 인해 시간이 아깝다는 생각이 들어요 / 그냥 사는 게 답답하고 몸과 마음이 따로 노는 것 같아요 / 작은 고민이 생기면 하루 종일 걱정을 해요 / 그럼에도 불구하고 책을 읽으셔야 해요 / 모든 답은 책에 있어요

3장 책이 주는 편안함
음악을 들으면서 책을 읽으면 왜 좋은가요? / 가족이 함께 글을 읽으면 더 도움이 되겠지요? / 밤늦게 책을 읽으면 안 되는 이유가 궁금해요 / 우울한 데도 책을 읽으면 도움이 되나요? / 어떤 날에 책을 읽으면 효과가 더 좋을까요? / 생각이 많아질 때도 독서가 좋을까요? / 독서가 생각을 바꿔 주는 구체적인 과정이 있나요?

4장 글자가 건네는 위로
1년 전만 해도 책 읽기가 너무 힘들었어요 / 안 되던 독서가 가능해졌어요 / 글을 읽으면 차분해지는 기분이에요 / 정신 건강을 위해서 책을 읽어야 한다는 말씀이 공감돼요 / 사막에서 빠져나온 것 같은 확신이 들어요 / 힘들 때 책을 읽거나 필사를 해요 / 창가에 앉아 나무와 꽃만 봐도 좋아요 / 책을 읽으면서 부드러워졌어요

5장 시가 건네는 위로
마음이 부서져 버렸어요. 이럴 때도 시가 도움이 될까요? / 자꾸 시를 읽으라고 하시는 이유가 궁금합니다 / 저는 시가 잘 읽히지 않아요 / 시가 좋다는 것은 알겠는데 손이 잘 안 가네요 / 나태주 시인의 〈맑은 하루〉라는 시가 저를 울렸어요 / 시를 읽다 보니, 시인의 시선이 부러워요 / 시집을 읽고 시를 쓰라는 말씀이 와닿았어요 / 매일 시 한 편은 읽으려고 노력해요

6장 글쓰기의 힘
여러분과 함께 걸어가는 시간 / 홀로 누리는 시간의 향기 / 간절히 그리운 봄 / 키보드 위에서 졸고 있는 생각 / 긴 안부 편지 〈일간 독서치료〉 / 흔들리는 나를 잡아 주는 글 / 글을 쓴다는 것 / 글쓰기는 고독한 질주
